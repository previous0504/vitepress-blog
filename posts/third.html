3.对数据进行劫持，initState初始化状态，options.data、methods、watch等，都可以初始化。

function initData(vm) {
    let data = vm.$options.data

    data = typeof data === 'function' ? data.call(vm) : data
  }

为什么要用data.call？为了让data中函数的this指向Vue实例。
